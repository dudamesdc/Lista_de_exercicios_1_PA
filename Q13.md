É quando a memória é alocada dinamicamente, mas não é liberada quando não é mais necessária.


### EXEMPLO 1:
#include <stdlib.h>
int main() {
    while (1) {
        int *ptr = (int *)malloc(sizeof(int)); // Aloca memória para um inteiro
        // Não libera a memória alocada
    }
    return 0;
}
